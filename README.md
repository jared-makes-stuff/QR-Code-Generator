# QR Code Generator

A fully customizable, privacy-first QR code generator built with React and Vite.

## Intent & Philosophy

There are many QR code generators available on the web today, but almost all of them come with frustrating gimmicks:

- **Limited Customizations**: They paywall basic features like colors, shapes, and logo uploads.
- **Privacy & Tracking Worries**: They inject their own redirect links into your QR codes to track scans, harvest data, or hold your links hostage.

**This project is different.**
This open-source QR code generator is designed to be completely free, highly customizable, and strictly private. The QR code behaves exactly as you intended, with no middlemen, no redirects, and no tracking.

## Features

- **No Tracking**: Direct linking only. Your data remains yours.
- **Limitless Customization**:
  - Customize QR code colors (background, dots, and corners).
  - Adjust the shape of the QR code dots and corners.
  - Upload a custom logo to center in the QR code.
- **Easy Exporting**: Download your generated QR codes instantly in `PNG`, `JPEG`, or `WEBP` formats.
- **Modern Tech Stack**: Fast and responsive UI built with Vite, React, and Tailwind CSS.

## Getting Started

1. Clone this repository:
   ```bash
   git clone https://github.com/jared-makes-stuff/QR-Code-Generator.git
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the development server:
   ```bash
   npm run dev
   ```
4. Build for production:
   ```bash
   npm run build
   ```
